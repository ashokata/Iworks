# InField Works Logo Setup

## Required Action

Please save the InField Works logo as:
- **File path:** `public/infield-works-logo.png`
- **Recommended dimensions:** Width: 200-400px, Height: proportional
- **Format:** PNG with transparent background preferred

The logo will be displayed in:
- Login page header
- Application manifest
- Browser tab (favicon)

## Current Logo References

The application is configured to use `/infield-works-logo.png` which maps to `public/infield-works-logo.png`.

Save your logo file to the `public` folder with this exact filename.
